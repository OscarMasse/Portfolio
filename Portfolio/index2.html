Gravity Cube

Unity GameDev GameJam C#

This is the first game I ever made with Unity! It was made during a 48h solo game jam. You can try it here! The days that followed, I created a database to store players high scores and a website to display it. The goal of the game is to protect the cube by destroying incoming asteroids with the three rotating comets. The game has been thought of as a mobile game, so the controls are very easy: the player can vary the comets' radius of rotation by touching the screen. Easy controls, but challenging gameplay nonetheless!

Abiding Game Jam

Unity GameDev GameJam C#

It was a game made during a small game jam, called Abiding Game Jam with people of the startup incubator Plaine Images. I was the game developer of a team of three. I worked with a game designer and a graphic artist. The theme was “the nonviolent communication” and we chose to create a game using player empathy in order to help snails find their appropriate shell, based on their personality.
